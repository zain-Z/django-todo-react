# Frontend

- javascript
- nodejs
- npm
- create-react-app
- yarn
- bootstrap
- reactstrap
- axios

## Getting started

To start project without getting any errors:

- Install pipenv by command: "python install pipenv" or "pip install pipenv"
- run "pipenv shell"
- run "python manage.py runserver"
- run "yarn start"
